# Evaluation Datasets

Keep fixed evaluation rows here.

## Dataset Rules

- Include happy-path, edge-case, and adversarial rows.
- Keep regression rows stable so baseline comparisons stay meaningful.
- Use JSONL unless the project has a stronger reason to use another format.
- Do not use training or fine-tuning data as evaluation data.

## Starter File

`regression.jsonl` is included as a tiny starter set and must be replaced or expanded for real project use.
