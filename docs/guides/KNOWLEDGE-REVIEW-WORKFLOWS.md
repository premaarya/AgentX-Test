# Knowledge And Review Workflows

## Purpose

This guide anchors the shared review, capture, and promotion workflow that AgentX surfaces in chat, commands, and sidebars.

## Review Loop

- Reuse prior learnings before planning or review.
- Produce durable review evidence before closeout.
- Capture promotable findings under `docs/artifacts/reviews/findings/` when follow-up should survive the current session.
- Create `docs/artifacts/learnings/LEARNING-<issue>.md` when the outcome should compound future work.

## Capture Rule

- Capture is mandatory for reusable workflow, architecture, review, or operator guidance.
- Capture is optional for narrow but still useful outcomes.
- Skip capture only when the outcome is trivial or already preserved elsewhere.
