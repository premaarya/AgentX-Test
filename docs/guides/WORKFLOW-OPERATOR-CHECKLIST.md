# Workflow Operator Checklist

Use this checklist when piloting the phase-one workflow surfaces for epic #215.

## Workflow Language

- Confirm the current checkpoint before acting: Brainstorm, Plan, Work, Review, Compound Capture, Done.
- Treat the next-step recommendation as advisory. It should never perform a hidden transition.

## Surface Map

- Command palette: `Show Workflow Next Step`, `Deepen Plan`, `Kick Off Review`, `Show Workflow Rollout Scorecard`, `Show Operator Enablement Checklist`
- Chat: `@agentx workflow next step`, `@agentx deepen plan`, `@agentx kick off review`, `@agentx rollout scorecard`, `@agentx enablement checklist`
- Sidebars: Work and Status both show the same workflow next-step contract

## Artifact Expectations

- Plan checkpoint: the issue should have an execution plan and a progress log.
- Work checkpoint: validation evidence should exist before review kickoff.
- Review checkpoint: review artifacts and durable findings should be linked.
- Compound Capture checkpoint: create the curated learning artifact or record the explicit skip rationale.

## Rollout Notes

- Workflow cohesion is the first pilot slice and must remain stable before later slices move forward.
- Use the rollout scorecard and pilot order guides for promotion, warning, or rollback decisions.
