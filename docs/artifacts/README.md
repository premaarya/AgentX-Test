# Artifact Families

This directory contains durable documentation artifacts that describe product intent, architecture, reviews, and reusable learnings.

## Canonical Locations

- `prd/` -> product requirements documents
- `adr/` -> architecture decision records
- `specs/` -> technical specifications
- `reviews/` -> review artifacts and durable findings
- `learnings/` -> reusable learning records

Legacy top-level paths under `docs/` remain as redirect-only shims to reduce migration risk while references are updated.
